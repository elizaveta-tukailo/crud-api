### Installing
1. Clone this repository   
2. Go to folder `crud-api`  
3. To install all dependencies use [`npm install`](https://docs.npmjs.com/cli/install)

---

### Running

```bash
# run application in development mode
$ npm run start:dev

# run application in production mode
$ npm run start:prod

# run test
$ npm run test
```
